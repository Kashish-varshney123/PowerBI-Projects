# Power BI Portfolio

This repository showcases my hands-on projects in data analysis and visualization using Power BI, SQL, and Excel. The focus of these projects is to transform raw data into meaningful insights and support data-driven decision-making.

---

## 📊 Projects

### 🛒 UrbanCart Sales Dashboard
- Analyzed 10,000+ retail transactions to identify sales trends and customer behavior  
- Built interactive dashboards with KPIs like Total Sales, Orders, and Average Order Value  
- Key Insight: Top-performing products and regions contribute the majority of revenue  

---

### 🏥 Aarogyam Hospital Dashboard
- Analyzed 900+ patient records to evaluate hospital costs and patient distribution  
- Developed dashboards to track treatment costs, patient trends, and insurance analysis  
- Key Insight: Identified high-cost departments and cost patterns for better decision-making
